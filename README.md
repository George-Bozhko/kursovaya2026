# Платформа поддержки специалиста по компьютерной экспертизе

Система автоматизированного анализа документов для выявления цифровых артефактов, оценки рисков и формирования экспертных отчётов.

## Описание проекта

Платформа предназначена для помощи специалисту по компьютерной экспертизе при первичном анализе электронных документов. Система автоматически выполняет поиск цифровых артефактов, оценивает уровень риска документа, проводит интеллектуальный анализ с использованием локальной языковой модели и формирует экспертный отчёт в формате PDF.

Проект разработан на языке Python с использованием Streamlit для создания веб-интерфейса и Ollama для локального запуска больших языковых моделей.

---

## Основные возможности

### Поиск цифровых артефактов

Система автоматически обнаруживает:

* Email-адреса
* IP-адреса
* Телефонные номера
* URL-адреса
* Доменные имена
* Пароли
* Логины
* MD5-хеши
* SHA1-хеши
* SHA256-хеши
* JWT-токены
* API-ключи

### Оценка риска

Для каждого документа вычисляется количественная оценка риска на основе весовой модели.

Уровни риска:

| Уровень     | Баллы |
| ----------- | ----- |
| Низкий      | 0–4   |
| Средний     | 5–9   |
| Высокий     | 10–14 |
| Критический | 15+   |

### Интеллектуальный анализ

Система использует локальные модели Ollama для:

* определения типа документа;
* оценки значимости документа;
* формирования краткого содержания;
* выделения подозрительных данных;
* подготовки экспертного заключения.

### Формирование отчётов

Автоматическое создание PDF-отчётов с результатами анализа:

* сведения о документе;
* найденные артефакты;
* оценка риска;
* результаты ИИ-анализа;
* экспертное заключение.

### Фильтрация и поиск

Поддерживается:

* поиск по содержимому документов;
* поиск конкретных артефактов;
* фильтрация по типам артефактов;
* отображение ключевых находок.

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/George-Bozhko/kursovaya2026.git
cd kursovaya2026
```

### 2. Установка зависимостей

```bash
py -m pip install -r requirements.txt
```

### 3. Установка Ollama

Установить Ollama:

https://ollama.com

Загрузить модель:

```bash
ollama pull qwen3:4b-instruct-2507-q4_K_M
```

## Запуск проекта

```bash
py -m streamlit run app.py
```

После запуска интерфейс будет доступен по адресу:

```text
http://localhost:8501
```
